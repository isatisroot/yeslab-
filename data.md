## 展示用户信息

访问方式：GET/userinfo/

请求参数：

| 参数   | 类型   | 是否必填 | 说明   |
| ------ | ------ | -------- | ------ |
| userid | 字符串 | 是       | 用户id |

用户信息不为空时，返回数据：

| 返回值 | 类型         | 说明                                 |
| ------ | ------------ | ------------------------------------ |
| name   | 字符串       | 用户的真实姓名                       |
| email  | 字符串       | 用户的邮箱地址                       |
| qq     | 字符串       | 用户的QQ号                           |
| phone  | 字符串       | 用户的手机号                         |
| adress | 字符串       | 用户的学校或企业名称                 |
| empty  | 布尔值(True) | 返回至前端，判断是否显示信息填写表单 |

| 返回值 | 类型          | 说明                                 |
| ------ | ------------- | ------------------------------------ |
| empty  | 布尔值(False) | 返回至前端，判断是否显示信息填写表单 |
